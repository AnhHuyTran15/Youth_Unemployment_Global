### Project Overview
This notebook showcases global youth unemployment using the datasets related to levels, distributions, rankings, and short-term forecasting of Africa. The workflow covers data loading, cleaning, exploratory analysis, visualization, and time‑series modeling.
### Data Description
The analysis uses youth_unemployment_global.csv, which contains 17,290 rows and 4 columns: Country, CountryCode, Year, and YouthUnemployment (percentage rate)
Environment and requirements
The notebook is written in Python 3.11 and uses the following key libraries: pandas, numpy, matplotlib, seaborn, and a forecasting library (Prophet-style API) for time‑series modeling. Ensure these dependencies are installed
